Cours à contenu variable

Les cours à contenu variable sont des cours dont le contenu varie d'un trimestre à l'autre.
Le sigle général de ces cours (qui se termine par X (ex. ADM100X)), est utilisé dans les descriptions officielles de programme. Le sigle général (ex. ADM100X) n'est jamais offert, contrairement aux sigles variables (ex. ADM100A, ADM100B, ADM100C).
Les descriptions des cours à contenu variable peuvent être consultées sur les sites des facultés, écoles, départements et programmes.
